Distributed Project

Distributed Tensorflow Training

A project built for distribute systems class. This project is designed to allow a group of people to assist witht the training of a tensorflow Deep Nueral Network. The model is supplied to all clients, and the training data is split into a queue to be supplied to the next clients request for data. Upon training the weights are sent back to the server to be recombined. follow the button for the detail report on this project
